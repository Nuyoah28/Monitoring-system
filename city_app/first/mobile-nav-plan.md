# 底部导航版手机端页面设计（4 分区）

## 分区规划
1. 总览（Home）
   - 实时监控主画面
   - 当前告警摘要
   - 今日运行指标

2. 智能（Agent）
   - Agent 对话
   - 快捷指令
   - 开放词汇加词

3. 设备（Devices）
   - 摄像头控制（云台/变焦/抓拍）
   - 实时环境（天气/AQI）
   - 设备状态列表

4. 我的（Profile）
   - 用户信息
   - 值班安排
   - 快捷设置与退出

## 设计要点
- 使用手机壳容器模拟 390x844 比例（接近常见手机）。
- 主内容区域滚动，底部导航固定。
- 单页多视图切换，后续可平滑迁移成 uni-app 的 4 个 tab 页面。
