# 业主端技能 (Owner Skills)

## 概述
业主端技能用于查询当前登录业主自己的服务数据。
所有涉及“我的”数据的查询都必须使用前端传入的用户 token，不查询其他业主的数据。

## 适用场景
- 业主查询个人资料、常驻区域、消息提醒状态。
- 业主查询社区提醒和系统消息。
- 业主查询自己的访客登记。
- 业主查询自己的报修记录。
- 业主查询停车场实时车位与车流量统计。
- 业主查询社区环境实时数据与趋势。

## 技能列表

### 1. get_owner_profile - 查询当前业主个人资料
- **功能**: 查询当前登录业主的个人资料。
- **参数**:
  - `none`: 无参数。
- **返回说明**:
  - 返回用户 ID、用户名、手机号、常驻区域、居民身份、消息提醒状态。

### 2. get_owner_messages - 查询当前业主社区提醒
- **功能**: 查询当前登录业主收到的社区提醒和系统消息。
- **参数**:
  - `limit`: 可选，返回条数，默认 `8`。
- **返回说明**:
  - 返回最近的消息内容与时间。

### 3. get_owner_visitors - 查询当前业主访客登记
- **功能**: 查询当前登录业主自己的访客登记记录。
- **参数**:
  - `limit`: 可选，返回条数，默认 `8`。
- **返回说明**:
  - 返回访客姓名、到访时间、车牌号。

### 4. get_owner_repairs - 查询当前业主报修记录
- **功能**: 查询当前登录业主自己的报修记录。
- **参数**:
  - `limit`: 可选，返回条数，默认 `8`。
- **返回说明**:
  - 返回设备名称、位置、报修时间、报修详情。

### 5. get_parking_realtime - 查询停车实时状态
- **功能**: 查询停车场实时车位状态。
- **参数**:
  - `monitor_id`: 可选，监控点 ID，默认 `1`。
  - `source`: 可选，`real` 或 `mock`，默认 `real`；真实数据为空时系统可降级为 mock 演示数据。
- **返回说明**:
  - 返回总车位、已占用、空余车位、占用率、分区车位情况。

### 6. get_parking_traffic_summary - 查询停车场车流量统计
- **功能**: 查询停车场今日和最近一次车流量统计。
- **参数**:
  - `monitor_id`: 可选，监控点 ID，默认 `1`。
  - `source`: 可选，`real` 或 `mock`，默认 `real`；真实数据为空时系统可降级为 mock 演示数据。
- **返回说明**:
  - 返回今日进出场、净流入、总流量，以及最近一次进出场统计。

### 7. get_environment_realtime - 查询社区实时环境
- **功能**: 查询指定监控点的实时环境监测数据。
- **参数**:
  - `monitor_id`: 可选，监控点 ID，默认 `1`。
- **返回说明**:
  - 返回温度、湿度、PM2.5、可燃气体、AQI 和更新时间。

### 8. get_environment_trend - 查询社区环境趋势
- **功能**: 查询指定监控点的环境趋势数据。
- **参数**:
  - `monitor_id`: 可选，监控点 ID，默认 `1`。
  - `range`: 可选，`day`、`week` 或 `month`，默认 `day`。
- **返回说明**:
  - 返回 AQI、湿度、PM2.5、可燃气体等趋势点。

## 调用示例
- `我的个人信息是什么` -> `get_owner_profile`
- `我有什么社区提醒` -> `get_owner_messages`
- `我的访客登记有哪些` -> `get_owner_visitors`
- `我最近提交了哪些报修` -> `get_owner_repairs`
- `现在还有多少空车位` -> `get_parking_realtime`
- `今天停车场车流量怎么样` -> `get_parking_traffic_summary`
- `社区环境怎么样` -> `get_environment_realtime`
- `今天空气质量趋势怎么样` -> `get_environment_trend`

## 安全边界
- 报修、访客、社区提醒、个人资料必须按当前登录 token 查询。
- 当前技能只做只读查询，不代替业主新增、修改或删除数据。
- 如果没有用户 token，应提示用户先登录业主端。
